# 🚀 Resume Builder AI

A modern AI-powered Resume Builder built with the MERN Stack that helps users create professional resumes, generate AI-assisted content, preview resumes in real time, save progress, and export resumes as PDF files.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Suraj1906/resume-builder-digital-heroes.git
cd resume-builder
```

---

## 🔧 Backend Setup

### Install Dependencies

```bash
cd server
npm install
```

### Create .env File

```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret

IMAGEKIT_PRIVATE_KEY=your_imagekit_private_key

OPENAI_API_KEY=your_gemini_api_key
OPENAI_BASE_URL=https://generativelanguage.googleapis.com/v1beta/openai/
OPENAI_MODEL=gemini-2.5-flash
```

### Run Backend

```bash
npm run server
```

---

## 🎨 Frontend Setup

### Install Dependencies

```bash
cd client
npm install
```

### Create .env File

```env
VITE_BASE_URL=http://localhost:3000
```

### Run Frontend

```bash
npm run dev
```
